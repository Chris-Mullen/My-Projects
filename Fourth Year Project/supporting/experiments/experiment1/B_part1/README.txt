To Run (in octave) type "noise" to generate the frames.
"mpg" is called from this function, which compresses the frame to video.
playback is enabled by default

default parameters:
noise(mkvid=true,play=true,frameout=false)

mkvid=true means compress frames to video
play=true means play the created video
frameout=true means decompress the video files into their componant frames
